Client to server

C2S_Alive -- Client fragt Server, ob dieser läuft/ verfügbar ist
C2S_AskKernel(Versions-Hash) -- Client stellt Anfrage an Server, ob die Kernel-Version veraltet ist
C2S_AskCore(Versions-Hash) -- Client stellt Anfrage an Server, ob die Core-Version veraltet ist
C2S_AskPackage(Paketname, Versions-Hash) -- Client stellt Anfrage an Server, ob die Pakete veraltet sind

C2S_GetKernel -- Client fragt Server nach URL zum Download der neuen Version
C2S_GetCore -- Client fragt Server nach URL zum Download der neuen Version
C2S_GetPackage(Paketname) -- Client fragt Server nach URL zum Download der neuen Version des angegebenen Pakets



Server to client

S2C_Alive -- Server antwortet Client, dass er läuft/ verfügbar ist
S2C_AnswerKernel(true/false) -- Server liefert dem Client ein true, falls eine neue Version verfügbar ist.
S2C_AnswerCore(true/false) -- Server liefert dem Client ein true, falls eine neue Version verfügbar ist.
S2C_AnswerPackage(true/false) -- Server liefert dem Client ein true, falls eine neue Version verfügbar ist.

S2C_SendKernel(URL) -- Server liefert dem Client eine URL zum Download der neuen Version
S2C_SendCore(URL) -- Server liefert dem Client eine URL zum Download der neuen Version
S2C_SendPackage(URL) -- Server liefert dem Client eine URL zum Download der neuen Version
