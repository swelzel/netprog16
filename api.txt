Client to server
C2S_Alive -- Client fragt Server, ob dieser läuft/ verfügbar ist
C2S_UpdateKernel(Mac-Adresse, Versions-Hash) -- Client stellt Anfrage an Server, ob die Kernel-Version veraltet ist
C2S_UpdateCore(Mac-Adresse, Versions-Hash) -- Client stellt Anfrage an Server, ob die Core-Version veraltet ist
C2S_UpdatePackage(Mac-Adresse, Versions-Hash) -- Client stellt Anfrage an Server, ob die Pakete veraltet sind

Server to client
S2C_Alive -- Server antwortet Client, dass er läuft/ verfügbar ist
S2C_UpdateKernel(true/false, URL) -- Server liefert dem Client ein true und eine Download-URL zurück, falls eine neue Version verfügbar ist.
S2C_UpdateCore(true/false, URL) -- Server liefert dem Client ein true und eine Download-URL zurück, falls eine neue Version verfügbar ist.
S2C_UpdatePackage(true/false, URL) -- Server liefert dem Client ein true und eine Download-URL zurück, falls eine neue Version verfügbar ist.
